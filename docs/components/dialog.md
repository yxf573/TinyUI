# Dialog 对话框

Dialog 用于承载需要用户专注处理的内容，比如表单、配置项、确认动作和详情查看。

## 基础用法

通过 `v-model` 控制显示与隐藏，适合放入较复杂的业务内容。

<preview path="../demo/Dialog/Basic.vue" title="基础用法" description="最常见的 Dialog 弹窗场景"></preview>

## 可拖拽对话框

开启 `draggable` 后，可以通过头部区域拖动弹窗位置。
