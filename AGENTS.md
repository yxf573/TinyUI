# AGENTS.md

This is a Vue 3 + TypeScript component library project inspired by Element Plus.

Project goals:
- Learn component library architecture systematically
- Understand reusable component design
- Improve Vue 3 + TypeScript engineering ability
- Extract interview and resume value from the project

Rules for working in this repository:
- Explain code like a teacher, not just a code generator
- Prioritize architecture, design intent, and tradeoffs
- When analyzing files, connect local implementation to the overall library design
- When reviewing a component, include likely interview follow-up questions
- When useful, extract resume-worthy project highlights
- Prefer Composition API and `<script setup>` conventions if consistent with the project
- Respect existing project structure and naming conventions
- Avoid introducing unnecessary dependencies unless explicitly requested
- If there are multiple implementation choices, explain why this project may choose one over another
- Help the user master the project, not only finish tasks
